Frequently Asked Questions

Standard operating questions for the NeuraSpend Expense Manager platform.

📊 How do I set a monthly budget? ▼

Navigate to the **Budget Settings** section via the dashboard or user dropdown. Set your Monthly Budget Limit and Warning Limit, then click "Apply Global System Preferences". If your monthly spending exceeds the warning limit, a warning alert badge will trigger!

🤖 How does Chitragupta AI work? ▼

Chitragupta is a highly contextual assistant powered by a Flask backend NLP parser. It automatically scans your database expense summaries, savings targets, and rewards tiers to provide real-time budget forecasting and audit summaries.

📷 How do I scan receipt images? ▼

Head to the **OCR Scanner** tab from the sidebar. Drag and drop any transaction receipt image (PNG, JPG) and hit "Start OCR Scan". The engine will extract the merchant name and total amount automatically, allowing you to auto-fill the expense ledger in a single click!

💎 How do I earn savings rewards? ▼

NeuraSpend rewards consistent savers! If your monthly spend stays under your monthly budget limit, you earn direct 2% cashback points at the end of the month. Accrue rewards to unlock Gold, Platinum, and Diamond Saver levels!

Troubleshooting Center

Resolve typical system errors and browser microphone configuration issues.

🎙️ My browser mic is not transcribing voice input ▼

Check that browser microphone permissions are enabled for NeuraSpend. The system relies on Web Speech API standard libraries. If you are on an offline system, please make sure your network supports localized speech engines, or type questions into Chitragupta's chat input directly!

📄 PDF financial statement report fails to compile ▼

Ensure your transaction database is not empty. If there are no expense records, the PDF rendering engine will throw empty bounds exceptions. Also verify your target directory path has valid folder write permission scopes.

👤 Suman is not able to add profile picture? ▼

Make sure your selected file is a valid image (PNG, JPG, JPEG, WEBP, GIF, SVG) and under 2MB. When uploading, click "Adjust Profile Settings" to submit. You will see a success confirmation dialog once saved!

Getting Started Onboarding

Follow these steps to master NeuraSpend on your first login.

1

Configure Budget Thresholds

Head over to the Budget Center. Initialize your monthly limit (default is ₹20,000) and set a warning threshold (default is ₹16,000). Applying preferences registers your monthly quotas instantly.

2

Record Expense Transactions

Log daily outflows via the add expense panel. You can either type transactions, scan raw physical receipts in the OCR tab, or dictate aloud using the Chitragupta AI microphone parsing service!

3

Track Savings Goals & Rewards

Create specialized savings targets (e.g. Vacation Fund, Emergency Fund). Credit savings at the end of the month to unlock simulated cashbacks and progress toward your saver milestones badges!

🛡️ SQL Injection & CSRF Defenses

NeuraSpend is built for enterprise security. Every SQLite query relies on parametrized queries to fully block SQL Injection. Every write request is intercepted by a centralized JS fetch filter to validate a unique UUID CSRF Token issued on load.

📁 Database Backups & Restorations

The Backup Manager allows hot-database backups. Click "Backup Database" to generate timestamped ZIP folders of the SQLite engine. You can restore prior states instantly via the administrative control terminal.
